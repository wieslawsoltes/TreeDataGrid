# Introduction

TreeDataGrid combines the strengths of a DataGrid and a TreeView into one control:

- **Flat mode** for tabular data presentation.
- **Hierarchical mode** for expandable tree rows with optional columns.
- **Strongly typed column model** (`TextColumn`, `TemplateColumn`, `CheckBoxColumn`, `HierarchicalExpanderColumn`).
- **Selection support** for rows and cells.

Use TreeDataGrid when you need large data sets displayed with columns and optional parent/child structure while staying within Avalonia control patterns.
