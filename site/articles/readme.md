# Documentation

This site covers TreeDataGrid from four angles:

- how to install the package and choose the right flat or hierarchical source model
- how columns, rows, selection, templating, and theme resources fit together
- how to customize, diagnose, and extend the control internals safely
- how to map public namespaces and generated API pages back to narrative docs
